# ランダミー
## ランダミーとは？
「今日はどの飲食店に行こうかな」と悩んでいませんか？その悩む時間、もったいないと思いませんか？  
そんな時に便利なのが「ランダミー」です。このアプリケーションは、あらかじめ登録しておいた飲食店の中からランダムに1店を選んでくれます。

## 機能
* アカウント登録
* あらかじめ登録した飲食店の中からランダムに1店を選択
* 使用したお金の管理

## 使用技術
* AWS
* HTML/CSS
* Bootstrap
* Ruby
* MySQL
* Ruby on Rails
* Heroku
